# Online Examination System
The Online Examination System is a web-based application developed to conduct examinations digitally in an efficient and secure manner. It enables administrators to create and manage exams, while students can take tests online with a fixed time limit. The system supports automatic evaluation and instant result generation, reducing manual effort and errors.
